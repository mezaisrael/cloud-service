# datacenter
CSCI Cloud Computing Project.

First install the required dependencies
```
$ npm install
```

After that, run the below commands to start the URB or datacenter service.

For URB
```$xslt
npm run urb&
```

For DataCenter
```$xslt
npm run datacenter&
```

To check if service is running:
```$xslt
ps aux | grep node
```
